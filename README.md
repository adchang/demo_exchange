# Demo Exchange
POC for an exchange written in C# and backed by SQLServer

## Features
- Currently supports Market and Limit orders
- Price-time priority
    - Buy limit price protection
- Quote, Level 2

## Roadmap
- Stop/Trailing Stop
- Order expiration
- Order Update/Cancel
